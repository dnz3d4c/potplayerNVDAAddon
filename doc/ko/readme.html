Daum pot player(자막/알림 자동 읽기)

  • 저자: aheu

Daum pot player의 표시되는 자막 및 알림 메시지를 NVDA가 자동으로 읽어주도록 합니다. 자막 및 알림 메시지가 pot player에 표시되게 하려면 설정> 접근성 메뉴에서 다음 항목들의 설정을 선택해야 합니다.

  • Context menu를 열고 설정 메뉴를 클릭합니다.

  • 접근성(TTS/UIA) 트리 뷰 항목을 찾은 다음 다음 항목이 선택되어 있는지 확인합니다:

    • 음성 출력 프로그램을 위해 메시지를 윈도우 제목으로 출력

    • 음성 출력 프로그램을 위해 자막을 윈도우 제목으로 출력

  • 확인 버튼을 눌러 설정을 저장합니다.
